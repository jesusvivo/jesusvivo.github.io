Marathon Performance Predictor

A multi-distance race-time predictor served end-to-end, on my own Garmin training data.

BentoML + ONNX behind a Feast feature store, deployed to GCP Cloud Run with scale-to-zero. Multisport fitness state (CTL / ATL / TSB across running, cycling, and swimming) feeds a personalized velocity-duration curve; running efforts are the labels. Evaluated per-distance with MAPE, never an averaged-minutes score. The headline output is a marathon finish-time projection with an uncertainty band, tracked across an 8-month block and validated against my Feb 2027 marathon debut.

F1 Pit Stop Prediction

Predicting when an F1 driver will pit next from per-lap telemetry.

Stacked XGBoost, LightGBM, and CatBoost with out-of-fold target encoding and a custom temporal CV splitter. Landed at 0.94 public LB ROC-AUC on Kaggle Playground Series S6E5 after seven iterations. The README has the full iteration log: what I tried, what failed, what worked.
